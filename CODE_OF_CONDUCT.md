# Code Of Conduct

## Our Pledge

We want AI Team Framework to be a practical, respectful, and useful open source project.

Participants are expected to communicate clearly, treat others with respect, and focus discussion on the work.

## Expected Behavior

- Be respectful and constructive.
- Prefer specific technical feedback over personal criticism.
- Keep issues and pull requests focused.
- Assume good intent while still asking for clarity when needed.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeatedly disrupting project discussion.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior conflicts with this code of conduct.

## Reporting

Use the repository maintainer contact channel or GitHub reporting tools when available.
