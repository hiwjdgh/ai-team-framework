# AGENTS

## Purpose

This project is developed with AI assistance to build AI Team Framework.

AI agents must read these documents first:

1. `memory/context.md`
2. `docs/product/overview.md`
3. `docs/architecture/system.md`
4. `docs/rules/coding-style.md`

## Development Rules

- Do not guess.
- Analyze existing code and documents first.
- Create reusable components and artifacts.
- Follow TypeScript strict mode for generated or implemented code.
- Do not add unnecessary libraries.

## Work Order

1. Analyze requirements.
2. Analyze related code and documents.
3. Analyze impact scope.
4. Implement.
5. Test.
6. Record changes.
7. Write history.
8. Write changelog.
9. Update current state.
